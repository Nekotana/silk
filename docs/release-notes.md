# Release Notes

## Silk v 0.15.2

### APIs introduced

- Battery
- Camera
- Input
- Lights
- Ntp
- Sensors
- Vibrator
- Wifi

### Devices supported
- Nexus 4 (Mako)
- Xiaomi Redmi Note 3 (Kenzo)
- QEmu on Linux and OS X

### Known issues:
- Initial device APIs are not overly consistent and subject to change
- API documentation is a work in progress
